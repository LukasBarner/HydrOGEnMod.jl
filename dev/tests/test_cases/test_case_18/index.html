Test Case 18

Test Case 18 is similar to Test Case 13, only that T_NLD is able to exercise market power in DEU. The tables below depict the sets and parameters, respectively. As the trader acts as a Cournot monopolist, the necessary capacity expansion in 2020 reduces to $0.5$. The test is passed, if only NLD_to_DEU and reverse is expanded by $0.5$ in 2020, while no other expansions take place. In addition, prices in DEU must be 0, 1.5 and 0.98 in 2020, 2025 and 2030, respectively, while those in NLD must be equal to zero. Passing criteria can be seen below, while the figure depicts a graphical solution.
